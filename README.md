# Stellar - Constellation Detection Platform

Stellar is a web-based platform that allows users to identify constellations in night sky images using advanced image processing and machine learning techniques. The platform also provides educational resources about astronomy and constellations.

## Features

- **Constellation Detection**: Upload night sky images and identify constellations
- **Comprehensive Library**: Browse through all 88 officially recognized constellations
- **Educational Resources**: Learn about astronomy and constellations through curated content
- **User Accounts**: Create an account to save your detections and track your learning progress
- **Responsive Design**: Access the platform from any device with a responsive user interface

## Pages

- **Home**: Landing page with an overview of the platform's features
- **Detection**: Upload images and identify constellations
- **Library**: Browse and search through the constellation database
- **Learning**: Access educational resources about astronomy and constellations
- **About**: Information about the platform and the team behind it
- **Login/Signup**: User authentication pages

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Image Processing**: Computer Vision algorithms for star detection
- **Machine Learning**: Constellation pattern recognition
- **Responsive Design**: Mobile-first approach for all device compatibility

## Getting Started

1. Clone the repository
2. Open `home.html` in your web browser to view the landing page
3. Navigate to the Detection page to try the constellation detection feature
4. Explore the Library and Learning sections to expand your knowledge
